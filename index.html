my-jeju-app.py
# 아이디어만 있으면 됩니다 🌊
import anthropic

def build_with_vibes(idea):
  client = anthropic.Anthropic()

  response = client.messages.create(
    model="claude-sonnet-4",
    messages=[{
      "role": "user",
      "content": idea
    }]
  )
  return response.content

# 3일 후, 실제 서비스로 배포됩니다 🚀
build_with_vibes("제주 맛집 추천 AI")

최신 AI 툴 대부분이 클라우드 기반이라 고사양 노트북이 필요하지 않습니다. 인터넷 연결이 가능한 노트북이면 충분합니다. OS는 Mac, Windows 모두 상관없으며, 사전에 필요한 프로그램 설치 가이드를 보내드립니다.

가능합니다. 1인실 추가 요금은 3박 기준 18만원이며, 신청서 제출 시 옵션으로 선택하실 수 있습니다. 선착순 4자리만 운영되니 빠르게 신청해주세요.

웹앱, 챗봇, 자동화 툴, API 서비스 등 다양한 형태가 가능합니다. 과거 참가자들은 AI 여행 플래너, 스마트 일정 관리 앱, 제주 맛집 추천 챗봇, 자동 이메일 분류기 등을 만들었습니다. 아이디어는 오셔서 발전시켜도 됩니다.
